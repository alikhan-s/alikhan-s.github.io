Cinema Ticket Booking Backend

A Spring Boot backend service implementing DAO and JPA/Hibernate. Provides a REST API (with Swagger) for managing theaters, movies, and users in PostgreSQL.

This project is a robust REST API built with Spring Boot. It serves as the backend for a hypothetical cinema ticket booking system.

Key Features:

  • RESTful Endpoints: Full CRUD operations for Cinemas, Movies, and Viewers.
  • Dual Data Approach: Demonstrates both the raw DAO (Data Access Object) pattern with JDBC and the modern JPA/Hibernate implementation.
  • Database: Connected and configured for PostgreSQL.
  • Tech Stack: Includes Java, Spring Boot (Web/JDBC), Maven, and PostgreSQL.
